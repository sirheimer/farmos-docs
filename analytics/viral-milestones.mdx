## Using Viral Data

When a post goes viral:

1. **Analyze why it worked**
   - What was the hook?
   - Which template?
   - What topic?
   - Which account type?

2. **Duplicate the winner**
   - Create similar content
   - Same template + different topic
   - Post to more accounts

3. **Scale the formula**
   - If "3 Ways" template went viral, create more "3 Ways" content
   - If faceless accounts outperform, create more faceless accounts
